# Seven Mile Startups - Prototype

A high-fidelity single-page website prototype for "Seven Mile Startups", a Cayman Islands-based entrepreneurship portal.

## Tech Stack
- **Vite**: Build tool and development server.
- **HTML5**: Semantic structure.
- **Tailwind CSS**: Utility-first styling.
- **Lucide Icons**: Modern, consistent iconography.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Features
- Responsive design (Mobile & Desktop).
- Interactive modals for "Interest" and "Sponsor" actions.
- "Cayman Professional" theme with Deep Caribbean Blue and Coral accents.
- Dynamic data handling for modals.
